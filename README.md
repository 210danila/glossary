# Glossary

__Инструкция по установке:__
1. Создаем вирутуальную среду и устанавливаем зависимости:
> make install
2. Активируем вирутальную среду у себя в терминале:
> source venv/bin/activate
3. Создаем базу данных и применяем миграции:
> make migrate
4. Запускаем скрипт:
> make start

Чтобы запустить скрипт на фоне выполняем следующую команду:
> nohup python app.py &

Чтобы удалить базу данных и заново ее создать:
> make refresh
*(Перед удалением создастся копия database_backup.py)*

Чтобы создать резервную копию базы данных:
> make backup
